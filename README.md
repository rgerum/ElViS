# ElViS - Elastiv Viscouse System Simulator

A javascript package to simulate systems of springs and dashpots.

You can find our playground here:
https://rgerum.github.io/ElViS/

Or take the interactive ElViS lesson to learn all about strings, dashpots and how they can be used to model biophysical systems:
https://rgerum.github.io/ElViS/question.html
